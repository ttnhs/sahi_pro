For Windows:

Download phantomjs from http://code.google.com/p/phantomjs/downloads/list
The current version is phantomjs-1.2.0-win32-dynamic.zip

Unzip phantomjs-1.2.0-win32-dynamic.zip inside this folder such that you have
sahi/ext/phantomjs/phantomjs.exe

----------------------------------------------------------------------------------------------------------------

For Linux: 

Open terminal and enter "sudo apt-get install phantomjs" command without quotation mark to install phantomjs.
