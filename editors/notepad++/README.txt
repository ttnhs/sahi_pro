For Autocomplete:
-----------------

1) Copy <SAHI_INSTALLED_DIRECTORY>\editors\notepad++\plugin\APIs\sahi.xml to <NOTEPAD++_INSTALLED_DIR>\plugin\APIs\


For Syntax highlighting (For Notepad++ lesser than version 6.4)
---------------------------------------------------------------

1) Go to View -> User-Defined Dialogue
2) Click Import
3) Choose <SAHI_INSTALLED_DIRECTORY>\editors\notepad++\sahi_syn.xml 
4) Restart Notepad++


For Syntax highlighting (For Notepad++ version 6.4 or above)
------------------------------------------------------------

1) Go to Language -> Define your language
2) Click Import
3) Choose <SAHI_INSTALLED_DIRECTORY>\editors\notepad++\sahi_syn.xml 
4) Restart Notepad++
